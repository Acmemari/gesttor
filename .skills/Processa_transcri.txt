INSTRUÇÃO PARA ANTIGRAVITY
Feature: Transcrição → ATA + Tarefas (com estilo do usuário)
🎯 OBJETIVO

Implementar uma funcionalidade onde o usuário cola uma transcrição de reunião (texto bruto) e o sistema automaticamente:

Interpreta o conteúdo
Gera um resumo executivo
Gera uma ATA estruturada
Extrai tarefas acionáveis
(Opcional) Salva tarefas no banco

⚙️ SERVICE PRINCIPAL

Criar /services/transcriptionProcessor.ts

import { buildTranscriptionPrompt } from "@/prompts/transcriptionPrompt"

export async function processTranscription(text: string) {
  const prompt = buildTranscriptionPrompt(text)

  const response = await callLLM({
    prompt,
    temperature: 0.2,
    maxTokens: 2000
  })

  try {
    const parsed = JSON.parse(response)
    return parsed
  } catch (error) {
    console.error("Erro ao parsear resposta:", error)
    throw new Error("Falha ao processar transcrição")
  }
}
